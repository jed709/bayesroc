# Extract Generated Stan Code

Returns the Stan program generated by
[`broc()`](https://jed709.github.io/bayesroc/reference/broc.md) for
inspection or manual use.

## Usage

``` r
get_stan_code(x)
```

## Arguments

- x:

  A `broc_model` object created by
  [`broc()`](https://jed709.github.io/bayesroc/reference/broc.md).

## Value

A character string of class `stan_code` containing the generated Stan
program.

## See also

[`get_stan_data()`](https://jed709.github.io/bayesroc/reference/get_stan_data.md)
for the matching data list;
[`get_numpyro_config()`](https://jed709.github.io/bayesroc/reference/get_numpyro_config.md)
for the equivalent artifact used by the JAX backend.

## Examples

``` r
if (FALSE) { # \dontrun{
model <- broc(brf(conf | old ~ cond), data = dat, family = evsd())
get_stan_code(model)
} # }
```
